WAREHOUSE DATA COLLECTOR - PORTABLE WINDOWS BUILD

1. Extract the complete WarehouseDataCollector folder.
2. Do not run the EXE from inside the ZIP.
3. Double-click WarehouseDataCollector.exe.
4. Keep the console window open while using the application.
5. The application opens in your default browser.

All configuration and data remain inside this portable folder.

Important folders:
  config\config.json
  data\manual_import\as400
  data\raw\as400
  data\AS400\AS400_Master.xlsx
  data\logs

Outlook features require Classic Outlook Desktop configured for the current
Windows user. New Outlook may not support COM automation. Manual AS400 import
remains available without Outlook.

No passwords are stored.
